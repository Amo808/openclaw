# Game Theory Basics

**Prisoner's Dilemma Payoff**:
| Player1 \ Player2 | Cooperate | Defect |
|-------------------|-----------|--------|
| Cooperate         | 3,3      | 0,5   |
| Defect            | 5,0      | 1,1   |

Nash: Both defect (Pareto suboptimal).

**Social Interactions**:
- Trust game: Sequential moves.
- Predict opponent type (e.g., paranoid defects first).

Matrix: Rows=your moves, Cols=his, cells=(your payoff, his).
Recommend dominant strategy.
